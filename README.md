# Game project 
 
 para correr el juego debes de seguir las siguientes intruscciones  en la terminar:

 ```sh
 git clone 
 cd game 
 ```
Activar entorno virtual windows
  ```sh
 cd game-env/Scrips 
 ./activate
 ```

 Activar entorno virtual windows
  ```sh
 source game-env/bin/activate
 ```

 desactivar env
  ```sh
 deactivate
 ```

 Ejecutar Programa
  ```sh
py main.py
 ```
